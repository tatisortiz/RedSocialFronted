## Local installation ğŸ› ï¸

### Backend

You can find the backend project here: [fdsProjectRedSocialMongo](https://github.com/tatisortiz/fdsProjectRedSocialMongo.git)


1. Clone the repository
   `$ git clone https://github.com/tatisortiz/fdsProjectRedSocialMongo.git`
2. Install dependencies
   `$ npm install --y`
3. Start Express on the server
   `$ npm run dev`
4. Run migrations
   `$ npm run db:refresh`

### Frontend

1. Clone the repository
   `$ git clone https://github.com/tatisortiz/RedSocialFronted.git`
2. Install dependencies
   `$ npm install --y`
3. Start Express on the server
   `$ npm run dev`
